    PRODUCTS Page

    This is the Products page where you can navigate by pointing and clicking your mouse or tabbing using your keyboard and enter key.

    ADD NEW PRODUCT click this option to open a new popup form where you can input item details.

    PRODUCT STATUS you can select the status from the dropdown list: Current/Active or Archived/Deleted.

    SHOW ENTRIES additional filter you can set based on number of records: 10, 25, 50, 100.

    SEARCH input a valid product description to filter, functionality work in progress.

    TABLE you can sort the records by clicking the UPC, Description, Supplier, Current Price, Current Quantity.

    EDIT click this option to navigate to another page where you can modify an existing item.

    ARCHIVE click this button to archive an item.

    PREVIOUS/NEXT you can select the paging based on number of available records.
