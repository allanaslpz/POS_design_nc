    ORDERS Page

    This is the Orders page where you can navigate by pointing and clicking your mouse or tabbing using your keyboard and enter key.

    CREATE NEW ORDER click this option to navigate to another page where you can enter the System Generated Code.

    ORDER STATUS you can select the status from the dropdown list: Open, Ordered, Received, All.

    SHOW ENTRIES additional filter you can set based on number of records: 10, 25, 50, 100.

    SEARCH input a valid order number to filter, functionality work in progress.

    TABLE you can sort the records by clicking the Order No, Sys Code, Order Date, Status headers.

    EDIT click this option to navigate to another page where you can modify existing order.

    REMOVE ITEM click this button to delete an item.

    PREVIOUS/NEXT you can select the paging based on number of available records.
