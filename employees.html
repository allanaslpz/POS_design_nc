    EMPLOYEES Page

    This is the Employees page where you can navigate by pointing and clicking your mouse or tabbing using your keyboard and enter key.

    ADD NEW EMPLOYEE click this option to open a new popup form where you can input employee details.

    SHOW ENTRIES additional filter you can set based on number of records: 10, 25, 50, 100.

    SEARCH input a valid employee name to filter, functionality work in progress.

    TABLE you can sort the records by clicking the Employee Name, Contact, City (hover mouse for complete address).

    EDIT click this option to navigate to another page where you can modify an existing employee.

    RESET PASSWORD click this option to send a change email notification to the selected employee.

    DELETE click this button to delete an employee.

    PREVIOUS/NEXT you can select the paging based on number of available records.
