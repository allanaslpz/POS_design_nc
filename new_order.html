    CREATE NEW ORDER

    This is the Create New Order page where you can navigate by pointing and clicking your mouse or tabbing using your keyboard and enter key.

    ORDER NUMBER the value is auto generated but it can be editable based on your records.

    SYSTEM GENERATED CODE this field holds the alt sys code from the old system.

    ORDER DATE this field defaults to the current system date, but it can be set by clicking on the calendar popup.

    ORDER STATUS you can set the value of this field by clicking the dropdown list with Order, Ordered, Received options.

    SAVE AND ADD PRODUCTS you can click this option to be able to Add Products on the same page.

    GO BACK TO ORDERS click this option to navigate back to the Orders page.
